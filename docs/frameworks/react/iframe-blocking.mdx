c15t provides two ways to block iframes:
- Headless: This is a headless implementation that does not render any DOM elements. It only manages the src/data-src attributes of existing iframe elements.
- Frame Component (React only): This is a React component that conditionally renders content and provides a pre-built/customizable placeholder. [Learn more about the Frame component](/docs/frameworks/react/components/frame)
