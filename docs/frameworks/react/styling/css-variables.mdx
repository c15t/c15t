## Overview

Every theme token is converted to a `--c15t-*` CSS custom property at runtime. You can override these variables in your own stylesheets without using the JavaScript theme API.

## Overriding Variables

Override in your stylesheet:

```css
:root {
  --c15t-color-primary: #8b5cf6;
  --c15t-color-surface: #fafafa;
  --c15t-radius-md: 1rem;
}

/* Dark mode overrides */
.dark {
  --c15t-color-primary: #a78bfa;
  --c15t-color-surface: #18181b;
  --c15t-color-text: #fafafa;
}
```

## Scoped Overrides

Target specific components by scoping variables:

```css
/* Only affect the consent banner */
[data-c15t="consent-banner"] {
  --c15t-color-surface: #f0f9ff;
  --c15t-radius-lg: 1.5rem;
}
```
