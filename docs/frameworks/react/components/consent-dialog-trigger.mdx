## Basic Usage

Use standalone or via the `showTrigger` prop on `ConsentDialog`:

```tsx
import { ConsentDialogTrigger } from '@c15t/react';

// Standalone
<ConsentDialogTrigger />

// Via ConsentDialog
<ConsentDialog showTrigger />
```
