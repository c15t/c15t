## Available callbacks

### onBannerFetched

Called when the consent banner is fetched; not invoked when the banner is in offline mode.

<AutoTypeTable path="./packages/core/src/types/callbacks.ts" name="OnBannerFetchedPayload" />

### onConsentSet

Called when the consent is set.

<AutoTypeTable path="./packages/core/src/types/callbacks.ts" name="OnConsentSetPayload" />

### onError

Called when an error occurs.

<AutoTypeTable path="./packages/core/src/types/callbacks.ts" name="OnErrorPayload" />

### onBeforeConsentRevocationReload

Called before the page reloads when consent is revoked. This only fires when `reloadOnConsentRevoked` is enabled (default: `true`) and the user revokes previously granted consent.

Use this callback to show a loading state or perform cleanup before the page reloads.
