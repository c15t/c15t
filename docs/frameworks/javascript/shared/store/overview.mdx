<section id="overview">
The store is a global state that can be used to access the consent manager's state, it includes information about the user's consent preferences, the location of the user, and the consent banner's state including methods to update them & more.
</section>

## Usage

```tsx
import { createConsentManagerStore } from 'c15t';

const store = createConsentManagerStore(consentManager);

const hasFunctionality = store.getState().has('functionality');
```
