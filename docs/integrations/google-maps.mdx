Google Maps loads through `@c15t/scripts` using c15t's script loader and the helper's default consent category.

## Adding Google Maps to c15t

<Callout type="info">
  See the [integration overview](/docs/integrations) for how to pass scripts to your framework (JavaScript, React, or Next.js).
</Callout>

```ts
import { googleMaps } from '@c15t/scripts/google-maps';

googleMaps({ apiKey: 'maps-key' })
```
