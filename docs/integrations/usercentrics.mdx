Usercentrics loads through `@c15t/scripts` using c15t's script loader and the helper's default consent category.

## Adding Usercentrics to c15t

<Callout type="info">
  See the [integration overview](/docs/integrations) for how to pass scripts to your framework (JavaScript, React, or Next.js).
</Callout>

```ts
import { usercentrics } from '@c15t/scripts/usercentrics';

usercentrics({ rulesetId: 'ruleset-123' })
```
