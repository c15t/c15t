Rybbit Analytics loads through `@c15t/scripts` using c15t's script loader and the helper's default consent category.

## Adding Rybbit Analytics to c15t

<Callout type="info">
  See the [integration overview](/docs/integrations) for how to pass scripts to your framework (JavaScript, React, or Next.js).
</Callout>

```ts
import { rybbitAnalytics } from '@c15t/scripts/rybbit-analytics';

rybbitAnalytics({ siteId: 'rybbit-123' })
```
