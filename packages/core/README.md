## Installation

```bash
npm install c15t
# or
yarn add c15t
# or
pnpm add c15t
```

## Quickstart 

```bash
# Generates the schema + code
npx @c15t/cli generate 
pnpm dlx @c15t/cli generate
yarn dlx @c15t/cli generate
bunx --bun @c15t/cli generate
```
