Headless Mode Custom UI

With data-headless="true", the embed skips rendering its own UI. The site builds a custom banner using the window.c15t API — like Zed does. 

<script src="c15t.js" data-backend="..." data-headless="true"></script>
<script>
  window.c15t.on('ready', () => {
    showMyCustomBanner();
  });
</script>
